## Run Project

To Run this project on your system :

Clone Project

```bash
  git clone REPO_LINK
```

Navigate to Project

```bash
  cd PROJECT_DIRECTORY
```

Install Dependencies

```bash
  npm install
```

Run Project

```bash
  npm run dev
```
